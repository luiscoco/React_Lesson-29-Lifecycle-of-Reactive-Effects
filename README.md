# Reactive Effects Playground (React 19 + Vite)

Learn the lifecycle of reactive effects (`useEffect`) hands-on.

## Quickstart
```bash
npm install
npm run dev
```

Then open the browser (it may auto-open) at the printed URL (default http://localhost:5173).

## Scripts
- `npm run dev` — start Vite dev server
- `npm run build` — production build
- `npm run preview` — preview the build

## What to try
- Switch **Server 1/2** to see cleanup + re-run.
- Change **User** to watch subscribe/unsubscribe.
- Toggle **Theme** and **Role** to see separate effects.
- Modify **Page Title** and check the browser tab title.
- Watch the in-app **Logs** panel for lifecycle messages.
